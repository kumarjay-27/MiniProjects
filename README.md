# Mini Projects Collection

This repository contains a collection of **C++ mini projects**. Each project demonstrates basic concepts of C++ programming, console-based applications, and simple game logic.
--

## Projects Included

### 1. Snake Game (C++ / Windows Console)
- A classic **Snake game** implemented in C++.
- Uses **`windows.h`** and **`conio.h`** for console input/output and frame control.
- Features:
  - Arrow key controls for the snake.
  - Randomly spawning fruits.
  - Score tracking.
  - Tail growth and collision detection.
- **Platform:** Windows (console application)

## How to Run
1. Open the project folder in an IDE that supports C++ (like **VS Code**, **Code::Blocks**, or **Visual Studio**).  
2. Compile the `.cpp` file.  
3. Run the executable.  
4. For the Snake game, use **arrow keys** to move the snake.
